# Pages Navigation Structure

### Main Routes

- `/` - **Home**
- `/prints-queue` - **Prints Queue**
- `/orders` - **Scheduled Orders**

### New Section

- `/new` (parent route)
    - `/new/print` - **Print**
    - `/new/order` - **Order**

### User Section

- `/user` (parent route)
    - `/user/history` - **History**
    - `/user/orders` - **Orders**

### Users Management

- `/users` (parent route)
    - **Admin Section**
        - `/users/admin/prints` - **Prints**
        - `/users/admin/orders` - **Orders**
    - **Shopkeeper Section**
        - `/users/shopkeeper/prints` - **Prints**
        - `/users/shopkeeper/orders` - **Orders**

### Informational Pages

- `/about-us` - **About Us**
- `/contact-us` - **Contact Us**
- `/policies` - **Policies**
